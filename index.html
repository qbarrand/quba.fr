About

 CV (PDF)

I like building large-scale web services using open-source stacks.

I am now working with CERN's Networking team on deploying a new campus-wide Wi-Fi network.
Before that, from 2013 to 2016, I held a Systems & Networks Engineer position at SNCF Réseau where I implemented web technologies on various real-time systems onboard IRIS 320, a special high-speed train dedicated to railway measurement.

Skills

Programming ...

Python 3 and Node.js are my prefered platforms for developping cross-platform applications.
On Windows, I have a good experience developping Windows Forms and ASP.NET / Nancy web apps with C#.
I used C on numerous occasions, mostly when performance was the key.

Systems ...

I have a sound knowledge of Entreprise Linuxes (RHEL, CentOS and Fedora) and Debian-based distributions.
At work, I used Puppet extensively to manage the configuration of network services.

Networks ...

I have a sound knowledge of Ethernet / IP networks, as well as a deep experience with large scale Wi-Fi and datacenter infrastructures. At CERN, I also contributed to several network automation projects.

Web ...

My understanding of the REST architectural style extends to the practical implementation of several HTTP web services and APIs.
Micro-frameworks in Python 3 and Node.js are my prefered environments, but I have an experience in PHP and C# / .NET too.
I am familiar with jQuery, AngularJS and CSS on the client-side.

Hobbies

I have an obvious but strong interest in Computer Science, open source software and radio networks. I am more generally passionate about science, especially astronomy, space conquest and particle physics.
I travel as much and as far as I can and happen to have a modest interest in photography; all the background pictures of this website were taken and edited by me, and I also post on Instagram from time to time.
